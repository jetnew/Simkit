A model is described by:
	init()
		Contains the following attributes:
			self.x_features
			self.y_features
			self.*params
			self.optimizer
			self.model = self.build()
	build()
		Defines the model structure, assigns self.model.
	loss(X, y)
		Defines the overall loss function for optimisation.
	fit(X, y, epochs=1000, verbose=1, plot=False, logdir="")
		Fits the model.
	predict(self, X)
		Predicts y given X.
